1. 웹 문서의 구조를 만드는 주요 시맨틱 태그에 관련된 설명이다.
다음 내용을 보고, 어떤 태그를 의미하는지 적으시오.
A. 핵심 콘텐츠를 담는 태그이며 웹문서에서 한번만 사용할 수 있는 태그 - body
B. 주로 맨 위쪽이나 왼쪽에 있으며 검색 창이나 사이트 메뉴를 삽입하는 태그 - header
C. 사이드 바 영역을 나타내는 태그 - aside
D. 사이트 제작정보나 저작권 정보, 연락처 등을 넣는 태그 - footer


2. html 문서의 기본 구조를 나타내는 태그 3가지를 적으시오.
html head body 


3. 슬랙에 올라온 이미지와 같이 테이블이 출력될수 있도록 html 코드를 작성하시오.
       <내용>
요일은 월, 화, 수만 표시하며, 교시는 1~3교시까지입니다.
표 제목은 "2학기 시간표"라고 작성하세요.
테이블과 셀의 테두리는 하나로 이어지게(border-collapse) 설정하세요.
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>Q3</title>
    <style>
        table{
            border-collapse: collapse;
            border: 1px solid black;
        }
        tr,td,th{
            border: 1px solid black;
        }
    </style>
</head>
<body>
    <table>
        <caption>2학기 시간표</caption>
        <tr>
            <th></th>
            <th>월</th>
            <th>화</th>
            <th>수</th>
        </tr>
        <tr>
            <td>1교시</td>
            <td colspan="2">국어</td>
            <td>수학</td>
        </tr>
        <tr>
            <td>2교시</td>
            <td>영어</td>
            <td>미술</td>
            <td rowspan="2">체육</td>
        </tr>
        <tr>
            <td>3교시</td>
            <td>사회</td>
            <td>과학</td>
        </tr>
    </table>
</body>
</html>


4. timetable.html 파일은 pages 폴더에 있습니다.
이 HTML 파일에서 images 폴더에 있는 table.png 이미지를 불러오려고 합니다.
아래 <img> 태그의 안을 올바르게 작성하세요.

my-website/
│
├── index.html
├── pages/
│   └── timetable.html
├── images/
│   └── table.png

정답 : <img src="../images/table.png"> 


5. 다음은 form 태그의 속성에 대한 설명이다. 
다음 내용을 보고, 어떤 속성을 의미하는지 적으시오.
A. 사용자가 입력한 내용을 서버 쪽 프로그램으로 어떻게 넘겨줄 것인지를 지정합니다. - method
B. form 태그 안에 작성된 내용을 처리해 줄 서버 프로그램을 지정합니다. 
C. 스크립트 파일을 현재 창이 아닌 다른 위치에서 열도록 합니다. - target
D. 자바스크립트로 폼을 제어할 때 사용할 폼의 이름을 지정합니다. - name


6. 슬랙에 올라온 이미지와 같이 설문 폼이 출력될수 있도록 html 코드를 작성하시오.

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8" />
  <title>간단한 설문 폼</title>
  <style>
    body {
      font-family: sans-serif;
      padding: 20px;
      color: #333;
    }
    label {
      display: block;
      margin: 8px 0 4px;
      font-weight: bold;
    }
    input[type="text"],
    textarea {
      width: 200px;
      padding: 6px;
      margin-bottom: 12px;
      box-sizing: border-box;
    }
    input[type="checkbox"], input[type="radio"] {
      width: auto;
      margin-right: 6px;
      display: inline-block;
    }
    .group-title {
      font-weight: bold;
      margin-top: 16px;
    }
    .checkbox-group label,
    .radio-group label {
      display: inline-block;
      font-weight: normal;
      margin-right: 12px;
      cursor: pointer;
    }
  </style>
</head>
<body>
  <form>
    <label for="name">이름:</label>
    <input>

    <p class="group-title">좋아하는 과일:</p>
    <div class="checkbox-group">
      <input type="checkbox" id="apple" name="fruit" value="ap">
        <label for="apple">사과</label>
      <input type="checkbox" id="banana" name="fruit" value="ba">
        <label for="banana">바나나</label>
      <input type="checkbox" id="orange" name="fruit" value="or">
        <label for="orange">오렌지</label>
    </div>

    <p class="group-title">성별:</p>
    <div class="radio-group">
      <input type="radio" id="male" name="gender" value="m">
        <label for="male">남자</label>
      <input type="radio" id="female" name="gender" value="f"> 
        <label for="female">여자</label>
    </div>
    <label for="opinion">의견:</label>
    <textarea></textarea> <br>
    <input type="submit"> 
    <input type="reset">
  </form>
</body>
</html>
